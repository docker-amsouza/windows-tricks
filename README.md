# 🖥️ Windows Tricks

Uma coleção de truques, comandos, atalhos e ferramentas úteis para **Windows 10** e **Windows 11**.

Este projeto tem como objetivo reunir, em um único lugar, comandos e dicas para facilitar o dia a dia de técnicos de informática, administradores de sistemas e usuários avançados.

---

## 📋 Conteúdo

Atualmente o projeto contém:

- 🎮 Reiniciar o driver de vídeo
- 📦 Comandos do Winget
- 🔧 Verificação de arquivos do sistema (SFC)
- 🛠️ Reparo da imagem do Windows (DISM)
- 💾 Verificação de disco (CHKDSK)
- 🔋 Relatório de bateria
- ⚡ Relatório de energia
- 🔑 Descobrir a chave OEM do Windows
- 🖥️ Informações do sistema
- 🌐 Limpar cache DNS
- 🔒 Verificar assinaturas digitais

Novos truques serão adicionados continuamente.
